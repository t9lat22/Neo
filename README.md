# Neo V1

Neo is an open source unblokced games website where people can fork it and make their onw unblocked games website.
---

## Important

To add more games you need to go to [Ultimate game stash](https://docs.google.com/document/d/1_FmH3BlSBQI7FGgAQL59-ZPe8eCxs35wel6JUyVaG8Q/edit?tab=t.0) then choose one after that copy the code into a new file in /g/ folder. For images you find game logos in Google.
